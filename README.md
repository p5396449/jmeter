# Performance Tests 

### Steps

1. Login (SSH) to your home directory from [GCP Console]()
2. You can clone the performance testing code from GitHub 
```bash
git clone <URL>
git checkout branchname
```
3. To start a test in GCP
```bash
cd branchName
./start_test.sh
```
4. To Stop a test in GCP
```bash
cd checkedOutBranch
./stop_test.sh 
```# jmeter
